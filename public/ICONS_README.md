# Ícones PWA

Para o PWA funcionar completamente, você precisa criar os seguintes ícones na pasta `public/`:

- `icon-192x192.png` - Ícone 192x192 pixels
- `icon-512x512.png` - Ícone 512x512 pixels

Você pode usar ferramentas online como:
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

Ou criar manualmente usando um editor de imagens.

O ícone deve representar o aplicativo de auditoria/hospitalar.
